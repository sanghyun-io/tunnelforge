Limited Import UI Policy Design

작성일: 2026-06-02

목적: Export가 성공했는데 limited_restorable이라는 이유만으로 사용자가 아무 Import도 할 수 없는 흐름을 고친다.

핵심 원칙: strict가 아닌 Dump를 권장 Import로 포장하지 않는다. 대신 사용자가 위험을 쉬운말로 이해하고 명시적으로 선택하면 제한적 Import를 허용한다.

사용자에게 보여줄 3단계 상태

상태 쉬운 설명 UI 동작 Rust Import 설정
strict_restorable 검증 증거가 충분한 Dump입니다. 권장 복원 경로로 진행할 수 있습니다. Import 버튼 활성화. 권장 Import로 표시. strict_manifest=true
limited_restorable Dump 파일은 사용할 수 있지만, 모든 데이터가 같은 순간의 스냅샷이라는 증거가 부족합니다. Import 버튼 활성화. 시작 전 제한적 복원 확인창 표시. strict_manifest=false
not_restorable 현재 버전의 TunnelForge가 안전하게 복원할 수 없는 Dump입니다. Import 버튼 비활성화. 차단 이유와 상세 설명 표시. Import 호출 안 함

not_restorable 기준

쉬운 설명: 이 Dump는 그대로 Import하면 대상 DB가 잘못 만들어질 가능성이 커서 프로그램이 막는 상태입니다.

대표 기준:

  • Dump manifest가 없거나 깨졌거나 지원하지 않는 형식입니다.
  • 필수 chunk 파일이 없습니다.
  • checksum이 맞지 않아 파일이 손상됐을 수 있습니다.
  • 프로그램이 모르는 기능이 manifest에 선언되어 있습니다.
  • object/schema 정보가 대상 DB와 맞지 않습니다.
  • Import하면 데이터가 틀어질 가능성이 큰 구조적 문제가 있습니다.

limited_restorable 선택 제공

쉬운 설명: Import 자체는 가능하지만, 완전 검증 Dump라고 말할 수는 없습니다.

예를 들어 threads=8로 Export하면 여러 DB 연결이 동시에 데이터를 읽습니다. 이 경우 모든 테이블이 정확히 같은 순간의 데이터인지 증명할 수 없습니다.

그래도 파일과 checksum은 정상일 수 있으므로, 테스트 복원이나 사용자가 위험을 받아들이는 제한적 복원은 가능하게 합니다.

제한적 Import 확인창 문구

제목: 제한적 복원으로 진행

이 Dump는 Import할 수 있지만 완전 검증 Dump는 아닙니다.

Export 중 원본 DB 데이터가 변경되었다면, 일부 테이블이 서로 다른 시점의 데이터일 수 있습니다.

완전 검증 복원이 필요하면 Export를 다시 실행해서 strict Dump를 만들어야 합니다.

테스트 복원, 제한적 이관, 또는 이 한계를 이해하고 받아들이는 경우에만 계속 진행하세요.

구현 범위

  • UI 호환성 영역에 쉬운말 상태 설명과 상세 이유를 함께 표시한다.
  • limited_restorable은 버튼을 활성화하되 제한적 복원 확인창을 거친다.
  • not_restorable은 버튼을 비활성화하고 우회 버튼을 제공하지 않는다.
  • Python wrapper에 strict_manifest 인자를 추가한다.
  • strict Dump는 strict_manifest=true, limited Dump는 확인 후 false로 호출한다.

테스트 기준

  • limited Dump에서 버튼이 활성화되는지 확인한다.
  • limited Dump 진행 시 확인창이 뜨고 strict_manifest=false가 전달되는지 확인한다.
  • not restorable Dump에서 버튼이 비활성화되는지 확인한다.
  • not restorable 기준 설명이 쉬운말로 노출되는지 확인한다.
  • Python wrapper가 strict_manifest=False를 Rust payload로 전달하는지 확인한다.

범위 밖

  • 병렬 Export에서 여러 연결이 하나의 MySQL transaction snapshot을 공유하게 만드는 기능.
  • Export thread 기본값 변경.
  • not_restorable 우회 Import.
