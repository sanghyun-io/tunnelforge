# Secrets Directory

TunnelForge desktop builds do not require a GitHub key, token, installation ID,
or PEM file. Keep this directory empty for normal client use.

Anonymous error reports are sent only after explicit consent through the
configured relay and contain the allowlisted fields described in
`docs/error_reporting.md`. Relay deployment credentials, if any, belong to the
relay operator and must never be placed in this repository or desktop config.
